# Ruleta de Saberes y Oportunidades

Aplicación estática, responsive y lista para GitHub Pages. La ruleta incluye estas categorías:

- Saberes del Territorio
- Intereses y Pasiones
- Redes de Apoyo
- Fortalezas Personales
- Oportunidades del Entorno

## Uso local

Abre `index.html` en tu navegador o sirve la carpeta con cualquier servidor estático.

## Publicación en GitHub Pages

1. Sube estos archivos a un repositorio de GitHub.
2. Entra en `Settings > Pages`.
3. Selecciona la rama y la carpeta raíz donde quedó `index.html`.
4. Guarda y espera la URL pública.

## Personalización

Si quieres cambiar textos, colores o agregar más categorías, edita `app.js`.
